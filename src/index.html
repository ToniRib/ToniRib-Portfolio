This project uses HTML5 Canvas and JavaScript to create an arcade game based on the classic Frogger. Udacity provided the art assets and the game engine. I coded the players, enemies, and other entities in JavaScript's object-oriented psuedo-classical style and added the unique objectie to get specific objects to target spaces.
